unmodifiable_collection library

Wrappers that prevent List, Set, or Map objects from being modified.

The Set and Map wrappers allow reading from the wrapped collection, but prohibit writing.

The List wrapper prevents changes to the length of the wrapped list, but allows changes to the contents.
